DATABASE MANAGEMENT STYSTEM

DBMS stands for Database management system. It is a software facilitating efficient data storage, retrieval and management in database
Ensure data safety and integrity, while offering accessibility and concurrency control.

  • data is information. It can take various forms, such as number, text, image,audio and video and can be stored in different formats. there are three types structured, semi-structured and unstructured data.
  • database is a container which can store data is called database
  • Management means how data are manage in database.

Data independance

Data Independence is mainly defined as a property of DBMS that helps you to change the database schema at one level of a system without requiring to change the schema at the next level. it helps to keep the data separated from all program that makes use of it.

We have namely two levels of data independence arising from these levels of abstracation:
  • Physical level data independence
  • Logical level data independence
  • Physical Level Data Independence :- It refers to the characteristic of being able to modify the physical schema without any alterations to the conceptual or logical schema, done for optimization purposes, e.g.the Conceptual structure of the database would not be affected by any change in storage size of the database system server. Changing from sequenoal to random access files is one such example. These alterations or Modifications to the physical structure my include:
    • Utilizing new storage devices.
    • Modifying data structures used for storage.
    • Altering indexes or using alternative file organization techniques etc.

  • Logical Level Data Independence:- It refers characteristic of being able modify the logical schema without affecting. the external schema or application program. The user view of the data would not be affected by any changes to the conceptual view of the data. These changes may include insertion or deletion of attributes,altering table structures entities or relationships to the logical schema, etc.
