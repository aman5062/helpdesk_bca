SETS

Sets are defined as "a well defined collections of object".Let's say we have a set of Natural Number(N) then it will have all the Natural Numbers as its member and the collection of the numbers is well defined that they are natural numbers.
NOTE: A Set is always denoted by a captial letters.
SOME COMMON SETS
Symbol Set Name Description Example
N Set of Natural Numbers The set of all the positive integers.Starts from 1 {1,2,3,4,..}
Z Set of Integers The set of all positive,negative,and Zero integers. {...,-3,-2,-1,0,1,2,3,...}
R Set of Integers The set of all real numbers,including positive and negative rational and irrational numbers. {..,-3.14,-2,-1,0,1,2,3.14,..}
C Set of Complex Numbers The set of all numbers that can be expressed a+bi,where a and b are real number and i is the imaginary unit (√(-1)). {1+2i,3-5i,√2,-πi,…}
Q Set of rational Numbers The set of all numbers that can be expressed as a fraction p/q,where p and q are integers and q The set of all numbers that can be expressed as a fraction p/q, where p and q are integers and q≠0. {1/2,/-3/4,5,0.75,..}
